# 对象

通过自定义字典来将枚举录入到系统中，菜单路径：自定义-列表。

点击右上角新增，添加一个新的字典：

分别录入字典名称，字典ID，以及子项，子项包括label，value，以及父项的value。

填写父ID字典可以形成一个具有父子关系的字典树，如省市区县的字典树，可用于自定义对象中的级联字段和多选级联字段。
