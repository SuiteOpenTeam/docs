# 菜单管理
在系统设置-用户和权限-菜单管理中配置系统菜单。

按照类别分为目录和菜单。

对于目录，选择上级菜单，填写菜单名称，选择菜单图标等信息。

菜单按类型分为3种：

- 自定义组件

- 关联自定义对象

- 外链

自定义组件适用于自定义前端页面，需要填写路由地址，组件路径。

关联自定义对象，选择绑定某一个对象，以及对象的某个视图，这样，当点击该菜单时，会跳转至该对象的视图表格页面。如果未绑定视图，跳转至默认视图。

外链，填写外链地址URL即可。

菜单添加完后，需要在角色管理里，分配菜单权限，该角色下的用户才能看到此菜单。
